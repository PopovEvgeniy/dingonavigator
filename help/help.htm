Dingo Navigator: User Reference

System requirements

Windows 98 or higher
Internet connection

Program description

 Dingo Navigator is a simple web browser with an intuitive user interface. This browser was made by Popov Evgeniy Alekseyevich. It uses the Trident engine.

Contact

Source code

The code is included in the source.zip archive. You need Turbo Delphi 2006 Explorer to compile it.

Third-party component

Dingo Navigator uses an icon set from the Tango Desktop Project. This icon set is in the public domain.

Program usage

You can interact with the browser via the menu items or via the toolbar buttons. You can see an example screenshot below.


Limitation

Multiple windows or tabs are not supported at this time.
