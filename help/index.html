Dingo Navigator: User Reference

System requirement

Windows XP or higher
Internet connection

Program description

Dingo navigator is a simple web browser with an intuitive user interface. This browser was made by Popov Evgeniy Alekseyevich.

Contact

You can e-mail me if you want.

Source code

The code is included in the source.zip archive. You need Turbo C++ 2006 Explorer to compile it.

Third-party component

Dingo Navigator uses an icon set from the Tango Desktop Project. This icon set is in the public domain.

Program usage

You can interact with the browser via menu items or toolbar buttons. You can see an example screenshot below.



You can work with browser tabs via the tab menu. Do right-click on the active tab title to open the tab's menu.
